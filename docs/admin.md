# Административные методы

## Выпуск внеочередного CRL

```
POST /api/ra/admin/crls/issue
```
- `authority` (string, nullable): Имя (псевдоним) издателя, на котором будет выпущен CRL. Если не задано, то CRL будет выпущен на издателе по умолчанию.
- `deltaOnly` (boolean): Выпустить только разностный CRL (без базового).

**Ответ:**
- 202: Команда на выпуск внеочередного CRL принята в обработку
- 400: Неверный запрос
- 403: Действие запрещено
